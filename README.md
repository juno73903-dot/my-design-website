# Portfolio Website (Static)

## Local preview
Open `index.html` in a browser.

## GitHub Pages deployment
1. Create a new GitHub repository.
2. Commit and push the contents of this folder.
3. In GitHub: Settings -> Pages -> Source: `Deploy from a branch`.
4. Select `main` branch and `/root` folder, then Save.
5. Wait for the Pages URL to appear.

## Updating assets
- Replace gallery images in `assets/` with your own files (keep filenames or update `data.js`).
- Replace `assets/resume.pdf` with your final resume PDF.
- Project links point to `project-1.html` ... `project-5.html` and are driven by `data.js`.
